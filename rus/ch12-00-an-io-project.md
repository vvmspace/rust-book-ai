# Проект ввода-вывода: Строим Command Line Tool

В этой главе мы напишем свой `grep` (утилиту для поиска текста).
Мы будем использовать всё, чему научились: векторы, строки, обработку ошибок, трейты, лайфтаймы и тесты.

Мы назовем его `minigrep`.

**План:**

1.  Принимать аргументы командной строки.
2.  Читать файлы.
3.  Рефакторить (потому что первый код всегда ужасен).
4.  Использовать переменные окружения.
5.  Писать в `stderr`.

Погнали писать свой велосипед!
