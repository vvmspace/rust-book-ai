# Комментарии

Комментарии — это то, что вы пишете, когда ваш код слишком сложен даже для вас.
Компилятор их игнорирует.

В Rust все стандартно: `//` для однострочных комментариев.

```rust
// Это комментарий
let lucky_number = 7; // И это тоже
```

Многострочных `/* ... */` обычно избегают, предпочитая несколько строк с `//`.

Есть еще документационные комментарии `///`, которые умеют превращаться в HTML-документацию, но о них поговорим в главе 14.

---

**Анекдот:**

> `// TODO: Переписать это, когда я пойму, как работают лайфтаймы.`  
> *(через 5 лет)*
> `// FIXME: Я все еще не понял, но оно работает.`
