# Установка бинарников с `cargo install`

С помощью `cargo install` можно устанавливать и использовать готовые инструменты, написанные на Rust.
Например, `ripgrep` (быстрый аналог grep) или `cargo-edit` (для команд типа `cargo add`).

```bash
cargo install ripgrep
```

Бинарники падают в `~/.cargo/bin`. Убедитесь, что этот путь есть в вашей `$PATH`.

Заметьте: это устанавливает софт из исходников (компилирует на вашей машине), если разработчик не предоставил пребилды (обычно cargo тянет исходники).

---

**JavaScript Analogy:**

> Это `npm install -g <package>`.
> Глобальная установка инструментов CLI.
