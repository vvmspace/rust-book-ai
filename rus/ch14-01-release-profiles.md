# Релиз-профили

В Rust есть профили компиляции, которые можно настраивать в `Cargo.toml`.
Два основных: `dev` (для `cargo build`) и `release` (для `cargo build --release`).

*   `dev`: Быстрая компиляция, мало оптимизаций (код медленнее), есть отладочная информация.
*   `release`: Долгая компиляция, агрессивные оптимизации (код быстрый), нет отладочной информации.

Можно настроить уровень оптимизации (`opt-level`) для каждого профиля:

```toml
[profile.dev]
opt-level = 1
```

Это позволяет гибко управлять балансом между скоростью сборки и скоростью работы программы.
