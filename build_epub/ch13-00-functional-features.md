# Функциональные фичи: Итераторы и Замыкания

Rust многое позаимствовал из функционального программирования.
В этой главе мы поговорим о:

1.  **Замыканиях (Closures)** — анонимных функциях, которые можно хранить в переменных.
2.  **Итераторах (Iterators)** — способах обработки последовательностей элементов.
3.  Как это делает наш `minigrep` быстрее и чище.

Спойлер: Итераторы в Rust быстрее, чем циклы `for`. Живите теперь с этим.

**English Joke:**

> Why do functional programmers hate side effects?
> Because they don't want to get involved.
