# Общие концепции программирования

Эта глава — как курс молодого бойца. Мы разберем концепции, которые есть почти в каждом языке программирования, но посмотрим на них через призму Rust.
Вы узнаете про переменные, базовые типы, функции, комментарии и управление потоком (if/else, циклы).

Это фундамент. Без него ваш дом (программа) рухнет при первом же `panic!`.
